HTML ELEMENTS

Tag Description Block, Inline, N/A Container or Empty
<!--..--> An HTML comment tag. Used to store comments, visible only in the HTML source. N/A Container
<!DOCTYPE> Lets the browser know what type of document to expect. N/A Empty
<a> An a tag is for anchoring a link to another page. Inline Container
<article> Specifies independent, self-contained content. Block Container
<aside> Content that is aside from the content its placed in. Block Container
<body> The body of an HTML document. N/A Container
<br /> A single line break. Inline Empty
<dd> A name tag for a description list. Block Container
<div> Used to define a division or section. Block Container
<dl> A description list. Block Container
<dt> A term or name in a description list. Block Container
<em> A phrase tag to give text emphasis. Inline Container
<fieldset> Used to group items in a form. Block Container
<figcaption> A caption for a <figure> element. Block Container
<figure> Used to define self-contained content like diagrams, photos, code listings, and more. Block Container
<footer> A footer for a document or section containing information related to its container. Block Container
<form> Used to create a form containing inputs, selectable elements, and more. Block Container
<h1> to <h6> Used for headings such as for titles in an HTML document. Block Container
<head> Contains head elements like scripts, styles, and meta information. N/A Container
<header> Usually contains navigational links. The top bar of a site. Block Container
<html> Informs the browser that the file is for an HTML document. N/A Container
<iframe> Used to embed other documents within an HTML document. Block and Inline Container
<img /> Used to add a picture to an HTML page. Inline Empty
<input> A field for a user to input data. Inline Empty
<label> A label for another element, such as an input or a button. Inline Container
<li> A list item. Block Container
<link /> Links the document to an external resource. N/A Empty
<meta /> Information about a page is put here and is parseable by browsers, search engines, and other web services. N/A Empty
<nav> Defines a set of navigation links. Block Container
<ol> An ordered list. Block Container
<p> A paragraph tag for displayed text. Block Container
<script> Either links to or directly provides client-side programs to be run on the page. Block and Inline Container
<section> Defines a section within a page. Block Container
<strong> A phrase tag that makes text more important. N/A Container
<style> Used to define style settings for an HTML page. N/A Container
<table> Creates a table to be filled with rows, columns, and cells. Block Container
<td> A cell in a table. N/A Container
<th> A cell at the top of a column in a table. N/A Container
<title> The title of the HTML page. N/A Container
<tr> A row in a table. N/A Container
<ul> An unordered list. Block Container
